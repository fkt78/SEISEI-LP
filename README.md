# SEISEI LP - 性格マッチング転職サービス

株式会社SEISEIの性格マッチング転職サービスのランディングページです。

## 🎯 機能

- **性格診断アプリ**: 32問の質問で8つのキャリアタイプを診断
- **レーダーチャート**: Chart.jsによる診断結果の可視化
- **コンタクトフォーム**: Formspree連携
- **レスポンシブデザイン**: PC・スマホ対応

## 🛠 使用技術

- HTML5
- Tailwind CSS (CDN)
- Font Awesome (CDN)
- Chart.js (CDN)
- Vanilla JavaScript
